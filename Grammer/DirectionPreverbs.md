# Direction Preverbs

According to their structure Direction Preverbs fall into simple and complex categories.

**Simple Direction Preverbs** show the direction of the doer’s action.

## Simple Direction Preverbs

| და         | to all directions (general)        |
|------------|------------------------------------|
| დავდივარ   | I am going, I go in all directions |
|            | სკოლაში დავდივარ ყოველ დღე.        |
|            |                                    |
| Imperative | იარე!/Go!                          |
|            | ჩქარა/ნელა იარე!                   |

| ა          | up                      |
|------------|-------------------------|
| ავდივარ    | I am going up, I go up  |
|            | ბავშვი ადის მარშუტკაში. |
|            |                         |
| Imperative | ადი!/Go up!             |
|            | ადი მარშუტკაში          |

| მი         | to the direction opposite the speaker |
|------------|---------------------------------------|
| მივდივარ   | I am going to, I go to                |
|            | მე მივდივარ სკოლაში.                  |
|            |                                       |
| Imperative | მიდი!/Go!                             |
|            | მიდი მაგიდასთან!                      |

| შე         | in/inside                    |
|------------|------------------------------|
| შევდივარ   | I am going in, I go (get) in |
|            | მოხალისე შედის გაკვეთილზე.   |
|            |                              |
| Imperative | შედი!/Go in!                 |
|            | შედი სახლში!                 |

| მო         | towards the speaker         |
|------------|-----------------------------|
| მოვდივარ   | I am coming, I come         |
|            | სად ხარ? ოფისიდან მოვდივარ. |
|            |                             |
| Imperative | მოდი!/Come!                 |
|            | მოდი ჩემთან!                |

| ჩა         | down                        |
|------------|-----------------------------|
| ჩავდივარ   | I am going down, I go down  |
|            | მოხალისე მარშუტკიდან ჩადის. |
|            |                             |
| Imperative | ჩადი!/Go down!              |
|            | ჩადი ავტობუსიდან!           |

| გა         | out                      |
|------------|--------------------------|
| გავდივარ   | I am going out, I go out |
|            | ქალი გადის მაღაზიიდან.   |
|            |                          |
| Imperative | გადი!/Go out!            |
|            | გადი აქედან!             |

| გადა       | over                        |
|------------|-----------------------------|
| გადავდივარ | I am going over, I go over  |
|            | მოსწავლეები გადადიან ხიდზე. |
|            |                             |
| Imperative | გადადი!/Go over!            |
|            | გადადი გადასასვლელზე!       |

| წა         | straight                                                                                |
|------------|-----------------------------------------------------------------------------------------|
| Note!      | The preverb წა has the imperative, <br> no form of conjugation in Present Simple Tense. |
|            |                                                                                         |
| Imperative | წადი!/Go!                                                                               |
|            | წადი სკოლაში!                                                                           |

### Preverb Cheat Summary

#### Present Simple Tense
- **და**ვდივარ (all directions)
- **მი**ვდივარ (opposite the speaker)
- **მო**ვდივარ (towards the speaker)
- **ა**ვდივარ (up)
- **ჩა**ვდივარ (down)
- **შე**ვდივარ (in) 
- **გა**ვდივარ (out)
- **გადა**ვდივარ (over or across)

#### Past Simple Tense
- **წა**ვედი (I went)
- **და**ვედი
- **მი**ვედი (I arrived)
- **მო**ვედი
- **ა**ვედი
- **ჩა**ვედი
- **შე**ვედი
- **გა**ვედი
- **გადა**ვედი

## Complex Directions Preverbs

Complex Direction Preverbs are formed from Simple ones by adding მო, 
and they mainly show the direction (of the action the doer does) towards the speaker.

| Simple | Complex |                                       |
|--------|---------|---------------------------------------|
| მი     | -       | to the direction opposite the speaker |
| მო     | -       | towards the speaker                   |
| ა      | ამო     | up                                    |
| ჩა     | ჩამო    | down                                  |
| შე     | შემო    | in / inside                           |
| გა     | გამო    | out / outside                         |
| გადა   | გადამო  | over (towards the speaker)            |
| წა     | წამო    | together with the speaker             |
| და     | -       | to all directions (general)           |

| Complex Preverbs Showing Directions                                                                                                                               |                                                                                                                                                                     |

(კიბეზე, სართულზე, მარშუტკიდან)                         |
