# Nature - ბუნება
| English  | ქართული |
| -------- | ------- |
| flower   | ყვავილი |
| tree     | ხე      |
| forest   | ტყე     |
| field    | მინდორი |
| mountain | მთა     |
| spring   | წყარო   |
| sea      | ზღვა    |
| lake     | ტბა     |
| river    | მდინარე |
| ocean    | ოკეანე  |

|         ტბა**ზე**         |
|       მდინარე**ზე**       |
