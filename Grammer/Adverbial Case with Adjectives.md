# Adverbial Case with Adjectives

To form the Adverbial Case, add **–ად/–დ**, an Adverbial Case marker at the end of the adjective.

If an adjective ends in **ი**, the final letter is dropped and **–ად** is added.
| Nominative Case   | Adverbial Case  |
| ----------------- | --------------- |
| კარგი (adjective) | კარგად (adverb) |
| good              | well            |

If an adjective ends in **ა**, **ე**, **ო**, or **უ**, add **–დ** in the Adverbial Case.
| Nominative Case   | Adverbial Case  |
| ----------------- | --------------- |
| საინტერესო        | საინტერესოდ     |
| interesting       | interestingly   |

**Note!** Adjectives in the Adverbial Case form adverbs and answer the question: **როგორ? how?** <br>
How are you? I am well. <br>
როგორ ხარ? კარგად.

**Note!** Adverbs and verbs are inseparable in the Georgian sentence. <br>
Giorgi writes well. <br>
გიორგი კარგად წერს. **or** გიორგი წერს კარგად.
